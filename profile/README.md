# 프로젝트 개요
### 서비스 이름
청년 상담 AI

### 한 줄 소개
청년의 이야기를 ‘기록’이 아니라 ‘이해’하는 AI.
음성 → 요약 → 상담 리포트까지 자동으로 완성하는 청년 특화 상담 도우미.

### 해결하려는 문제
- 상담사는 매 상담마다 기록을 수기로 남겨야 함
- 상담 이후의 정리·분류·보고 과정이 부담
- 상담 데이터는 쌓이지만, 정책 추천·위기 탐지 등 후속 활용이 어려움
- 청년의 이야기가 문서화되는 과정에서 정보 손실
