## 팀 소개 👋

``` “우린 쉰 게 아니라, 뭘 했던 사람들이다.”```

‘뭘했음 청년’은 일시적인 경력 공백을 ‘쉼’이 아닌 ‘경험의 시간’으로 바라보는 새로운 관점을 담은 이름입니다.
‘쉬었음 청년’이 멈춤과 결핍을 전제로 했다면, ‘뭘했음 청년’은 그 시간 동안의 무엇이든 한 경험, 도전을 존중합니다.
이 팀명은 사회가 규정한 스펙의 틀을 넘어, 모든 청년의 시간에는 의미가 있다는 메시지를 전합니다.
‘쉬었음’이 아니라 ‘뭘했음’으로 정의하여, 대회에 임하고자 합니다.
